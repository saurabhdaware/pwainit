# PWAinit
Initiate PWA project and get boilerplate code for Progressive Web App features or Turn your existing website into Progressive Web App with `npm i -g pwainit` ðŸŽ‰

PWAinit lets you intialize your PWA development process without writing basic service worker, manifest, push api and index.html code.

## Install
```
npm install -g pwainit
```
## Create new project
```
pwainit create helloworld
```

### Turn existing site to PWA
```
cd existingWebsite
pwainit add
```
